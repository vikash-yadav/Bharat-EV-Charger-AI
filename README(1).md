# ⚡ BharatCharge Connect

> **UPI for EV Charging in India**

BharatCharge Connect is a next-generation EV charging interoperability platform designed to create a connected, reliable, and seamless charging ecosystem across India.

## 🚀 Features

- Universal Charger Discovery
- Real-Time Charger Availability
- Smart Route Planning
- AI Charging Assistant
- BharatCharge Pay (Unified Payments)
- Fleet Management
- Charging Analytics Dashboard
- Cross-Network Interoperability

## 🛠️ Tech Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- ShadCN UI

### Backend
- Firebase Authentication
- PostgreSQL
- REST APIs

### AI
- Gemini AI

### Maps & Navigation
- Google Maps API

## 🚀 Getting Started

```bash
git clone https://github.com/your-username/bharatcharge-connect.git
cd bharatcharge-connect
npm install
npm run dev
```
